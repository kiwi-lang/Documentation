Detailed Description

Authority decides which player can see which convo

// On Authority GameState->Manager->NewConversation("Global", 0) // 0 is always global GameState->Manager->NewConversation("Team1", 1) GameState->Manager->NewConversation("Team2", 2)

for Controller in team1: GameState->Manager->AuthorizeActor(Controller, 1)

for Controller in team2: GameState->Manager->AuthorizeActor(Controller, 2)

// On client GameState->Manager->Send(Controller, "Team", FGKChatEntry{...})
