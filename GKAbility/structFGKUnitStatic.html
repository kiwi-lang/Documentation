Detailed Description

Character/class specific data

/rst

.. note::

Attribute set can be initialized by the ASC from a datatable using virtual void UAttributeSet::InitFromMetaDataTable(const UDataTable* DataTable). We could specialize this method to use our unit datatable instead.

The default iplementation expects the row to be named "<OwnerName>.<AttributeName>". In our case OwnerName means nothing because of our data driven design we do not have one Blueprint/C++ class per type of unit

/endrst
